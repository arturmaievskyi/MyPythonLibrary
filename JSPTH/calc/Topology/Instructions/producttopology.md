## Overview

### What is a Product Topology?

A **product topology** is a topology on the Cartesian product of multiple topological spaces. Given spaces $(X_1, \tau_1), (X_2, \tau_2), \ldots, (X_n, \tau_n)$, the product topology on $X_1 \times X_2 \times \cdots \times X_n$ is generated by the basis:

$$\mathcal{B} = \{U_1 \times U_2 \times \cdots \times U_n : U_i \in \tau_i\}$$

### Key Characteristics

| Property | Description |
|----------|-------------|
| **Open sets** | Unions of products of component opens |
| **Basis** | $U_1 \times U_2 \times \cdots \times U_n$ where $U_i$ open in $X_i$ |
| **Projections** | Maps $\pi_i: X_1 \times \cdots \times X_n \to X_i$ are continuous and open |
| **Separation** | Product of Hausdorff spaces is Hausdorff ✓ |
| **Compactness** | Product of compact spaces is compact (Tychonoff's theorem) ✓ |
| **Connectedness** | Product is connected iff all components are connected |
| **Metrizability** | Finite product of metrizable spaces is metrizable ✓ |

### When to Use

✅ **Perfect for:**
- Multi-dimensional topological problems
- Studying continuous functions into products
- Understanding inherited properties
- Analyzing convergence in multiple variables
- Building complex spaces from simpler ones
- Studying compact spaces (Tychonoff)

❌ **Not ideal for:**
- Purely linear problems
- Spaces with very different dimensions
- Compatibility issues between spaces

## Mathematical Background

### Definition

For topological spaces $(X_i, \tau_i)$ where $i = 1, 2, \ldots, n$, the **product topology** on $X = X_1 \times X_2 \times \cdots \times X_n$ is the coarsest topology making all projection maps $\pi_i$ continuous.

**Basis for product topology:**
$$\mathcal{B} = \{U_1 \times U_2 \times \cdots \times U_n : U_i \in \tau_i \text{ for each } i\}$$

**Open sets:** Unions of basis elements.

### Key Example: $\mathbb{R}^2$

Consider the Euclidean plane $\mathbb{R}^2 = \mathbb{R} \times \mathbb{R}$:

```
Open sets include:
  ∅, ℝ², open rectangles (a,b)×(c,d),
  unions of open rectangles, etc.

Product topology on ℝ² is the same as standard Euclidean topology!
```

### Fundamental Theorems

1. **Projection Continuity**: All projection maps $\pi_i: X_1 \times \cdots \times X_n \to X_i$ are continuous and open.

2. **Inherited Separation Axioms**:
   - Product is T₀ ⟺ all components are T₀
   - Product is T₁ ⟺ all components are T₁
   - Product is T₂ (Hausdorff) ⟺ all components are T₂

3. **Tychonoff's Theorem**: Product of compact spaces is compact (even infinite products!)

4. **Connectedness**: Product is connected ⟺ all components are connected

5. **Separability** (finite products): Product of separable spaces is separable

6. **Metrizability** (finite products): Finite product of metrizable spaces is metrizable

```python
from indiscrete_topology import IndiscreteTopology
```

### Open and Closed Sets

Open sets in the product are generated by products of component opens.

```python
from product_topology import ProductTopology
from indiscrete_topology import IndiscreteTopology

space1 = IndiscreteTopology({1, 2})

test_set = {(1, 'a'), (1, 'b')}

print(f"Open sets: {len(product.open_sets)}")
for open_set in product.open_sets[:5]:  # Show first 5

clopen = product.get_all_clopen_sets()
```

### Interior, Closure, Boundary

```python
subset = {(1, 'a'), (2, 'b')}

interior = product.interior(subset)
print(f"Interior: {interior}")

closure = product.closure(subset)
print(f"Closure: {closure}")

boundary = product.boundary(subset)

# Verify: boundary = closure - interior
print(f"Correct: {boundary == (closure - interior)}")
```

## Projections

### What are Projections?

A **projection map** $\pi_i: X_1 \times X_2 \times \cdots \times X_n \to X_i$ extracts the $i$-th coordinate:

$$\pi_i(x_1, x_2, \ldots, x_n) = x_i$$

### Basic Projections

```python
# Project onto specific coordinate
proj_0 = product.project_onto_coordinate(0)
print(f"π₀(X × Y) = {proj_0}")  # Set of first coordinates

proj_1 = product.project_onto_coordinate(1)
print(f"π₁(X × Y) = {proj_1}")  # Set of second coordinates

# Project onto multiple coordinates
proj_both = product.project_onto_coordinates([0, 1])
print(f"π₀₁(X × Y × Z) = {proj_both}")
```

### Projection Maps (Functions)

```python
# Get projection map π₀
pi_0 = product.projection_map(0)

# Use it
point = (1, 'a')
first_coord = pi_0(point)
print(f"π₀({point}) = {first_coord}")

# All projections are continuous
is_cont = product.is_projection_continuous(0)
print(f"π₀ is continuous? {is_cont}")  # Always True

# All projections are open
is_open = product.is_projection_open(0)
print(f"π₀ is open? {is_open}")  # Always True
```

### Projections of Subsets

```python
subset = {(1, 'a'), (1, 'b'), (2, 'a')}

# What coordinates appear?
coord0_values = product.project_onto_coordinate(0, subset)
coord1_values = product.project_onto_coordinate(1, subset)

print(f"First coordinates: {coord0_values}")   # {1, 2}
print(f"Second coordinates: {coord1_values}")  # {'a', 'b'}
```

## Separation Axioms

### Hausdorff Property

The most important: **Product is Hausdorff iff all components are Hausdorff**.

```python
space1 = DiscreteTopology({1, 2})  # Hausdorff
space2 = DiscreteTopology(['a', 'b'])  # Hausdorff
product = ProductTopology([space1, space2])

# Product inherits Hausdorff
is_hausdorff = product.is_t2_hausdorff()
print(f"Is Hausdorff? {is_hausdorff}")  # True

# Separating points with disjoint neighborhoods
p1 = (1, 'a')
p2 = (2, 'b')

nbhds_p1 = product.open_neighborhood(p1)
nbhds_p2 = product.open_neighborhood(p2)

for n1 in nbhds_p1:
    for n2 in nbhds_p2:
        if not (n1 & n2):  # Disjoint
            print(f"Separating neighborhoods found")
            break
```

### All Separation Axioms

```python
axioms = product.separation_axioms()

print("Separation axioms:")
for axiom, satisfied in axioms.items():
    symbol = "✓" if satisfied else "✗"
    print(f"  {symbol} {axiom}: {satisfied}")

# Inheritance rules:
# T₀ ⟹ inherited
# T₁ ⟹ inherited
# T₂ (Hausdorff) ⟹ inherited
# T₃ (Regular) ⟹ inherited (if components are)
# T₄ (Normal) ⟹ NOT always inherited for infinite products
```

### Properties of Component Spaces

```python
def check_separation_inheritance(product):
    """Check what separations inherit."""
    print(f"\n{'='*60}")
    print("Separation Inheritance Analysis")
    print(f"{'='*60}")
    
    component_props = {
        'T0': all(s.is_t0() for s in product.spaces),
        'T1': all(s.is_t1() for s in product.spaces),
        'Hausdorff': all(s.is_t2_hausdorff() for s in product.spaces),
    }
    
    product_props = product.separation_axioms()
    
    for prop in ['T0 (Kolmogorov)', 'T1', 'T2 (Hausdorff)']:
        short = prop.split()[0].rstrip('(')
        comp_val = component_props.get(short.rstrip('('))
        prod_val = product_props.get(prop)
        
        match = "✓" if comp_val == prod_val else "✗"
        print(f"{match} {prop}: Components={comp_val}, Product={prod_val}")

check_separation_inheritance(product)
```

## Connectivity

### Is Connected?

**Key theorem:** Product is connected iff all components are connected.

```python
from indiscrete_topology import IndiscreteTopology

# Both components connected
space1 = IndiscreteTopology({1, 2, 3})  # Connected
space2 = IndiscreteTopology(['a', 'b', 'c'])  # Connected
product = ProductTopology([space1, space2])

is_connected = product.is_connected()

# One component disconnected
space3 = DiscreteTopology({1, 2})  # Disconnected (singletons)
product2 = ProductTopology([space1, space3])

is_connected2 = product2.is_connected()
print(f"Is connected? {is_connected2}")  # False
```

## Compactness & Tychonoff

### Tychonoff's Theorem

**Key theorem:** Product of compact spaces is compact.

This is true even for infinite products! This is one of the most important theorems in topology.

```python
# All components compact
space1 = IndiscreteTopology(range(5))  # Finite = compact
space2 = IndiscreteTopology(['a', 'b', 'c'])  # Finite = compact
product = ProductTopology([space1, space2])

is_compact = product.is_compact()
print(f"Is compact? {is_compact}")  # True

# Verification: finite spaces are always compact
print(f"Size: {product.size}")  # Finite
print(f"Number of open sets: {len(product.open_sets)}")  # Finite

# Can cover with finitely many open sets easily
```

## Convergence in Products

### Component-wise Convergence

**Key theorem:** Sequence $(x_n)$ converges to $x$ in product iff it converges in each component:

$$(x_n) \to x \text{ in } X_1 \times X_2 \iff \pi_i(x_n) \to \pi_i(x) \text{ in each } X_i$$

```python
# Define sequence in product
sequence = [(1, 'a'), (1, 'b'), (1, 'a'), (1, 'b')]

# Check convergence in product
limit_point = (1, 'a')
converges = product.converges_to(sequence, limit_point)
print(f"Sequence converges to {limit_point}? {converges}")

# Check by components
component_convergence = product.component_convergence(sequence, limit_point)
print(f"Converges by component analysis? {component_convergence}")

# These should match
print(f"Methods agree? {converges == component_convergence}")
```

### Limit Points of Sequences

```python
sequence = [(1, 'a'), (2, 'b'), (1, 'c'), (2, 'a')]

# Get all limits
limits = product.get_limits(sequence)
print(f"All limits: {limits}")

# For each limit point
for point in limits:
    projects_0 = product.project_onto_coordinate(0, {point})
    projects_1 = product.project_onto_coordinate(1, {point})
    print(f"  {point}: first coord = {projects_0}, second = {projects_1}")
```

## Continuous Functions

### Functions into Products

A function $f: X \to Y_1 \times Y_2 \times \cdots$ is continuous iff each component function is continuous:

$$f = (f_1, f_2, \ldots) \text{ continuous} \iff f_i: X \to Y_i \text{ continuous for all } i$$

```python
product = ProductTopology([space1, space2])

def f(x):
    """Function from some space into product."""
    return (x * 2, chr(ord('a') + x))

# Check continuity
target = product
is_continuous = product.is_continuous_function(f, target)
print(f"Is continuous? {is_continuous}")
```

### Projection Continuity

```python
# Projection maps are always continuous
for i in range(product.dimensions):
    is_cont = product.is_projection_continuous(i)
    print(f"π_{i} is continuous? {is_cont}")  # Always True

# Projections are also open maps
for i in range(product.dimensions):
    is_open = product.is_projection_open(i)
    print(f"π_{i} is open? {is_open}")  # Always True
```

## Special Maps & Diagonals

### Diagonal Map

The **diagonal map** $\delta: X \to X \times X$ sends $\delta(x) = (x, x)$.

```python
# Create X × X
space = IndiscreteTopology({1, 2, 3})
product_xx = ProductTopology([space, space], name="X × X")

# Get diagonal map
diagonal = product_xx.diagonal_map()

# Use it
x = 1
image = diagonal(x)
print(f"δ({x}) = {image}")  # (1, 1)

# Get diagonal set
diagonal_set = product_xx.diagonal_set()
print(f"Diagonal set: {diagonal_set}")
# {(1,1), (2,2), (3,3)}
```

### Diagonal Properties

```python
# Is diagonal closed?
is_diag_closed = product_xx.is_diagonal_closed()
print(f"Is diagonal closed? {is_diag_closed}")

# Is diagonal open?
is_diag_open = product_xx.is_diagonal_open()
print(f"Is diagonal open? {is_diag_open}")

# In Hausdorff spaces, diagonal is always closed
# In indiscrete spaces, diagonal is usually dense
```

### Importance

The diagonal map is important for:
1. Detecting separation axioms (Hausdorff ⟺ diagonal is closed)
2. Studying uniform structures
3. Analyzing diagonal topology

## Best Practices

### ✅ DO

1. **Check component properties first**
   ```python
   if all(s.is_t2_hausdorff() for s in product.spaces):
       print("Product is Hausdorff (inherited)")
   ```

2. **Use Tychonoff for compactness**
   ```python
   if all(s.is_compact() for s in product.spaces):
       print("Product is compact (Tychonoff)")
   ```

3. **Understand projection maps**
   ```python
   pi_0 = product.projection_map(0)
   image = pi_0(point)  # Continuous and open
   ```

4. **Use component convergence**
   ```python
   is_conv = product.component_convergence(seq, limit)
   # More efficient than checking neighborhoods
   ```

5. **Document product structure**
   ```python
   # X × Y × Z: connecting spaces
   # X = points, Y = colors, Z = times
   ```

### ❌ DON'T

1. **Don't assume infinite product properties**
   ```python
   # ❌ Infinite products may not be metrizable
   # even if components are
   ```

2. **Don't confuse product with other operations**
   ```python
   # ❌ Product ≠ coproduct (disjoint union)
   # Product = Cartesian product with induced topology
   ```

3. **Don't forget Tychonoff**
   ```python
   # ❌ Compactness DOES inherit in products
   # (This is exceptional - most properties don't inherit universally)
   ```

4. **Don't ignore dimension**
   ```python
   # ❌ dim(X × Y) = dim(X) + dim(Y) in many cases
   # But need to be careful about definition
   ```

5. **Don't mix up slices and sections**
   ```python
   # ✓ Slice: fix one coordinate
   # ✓ Section: fix multiple coordinates
   # They're related but different
   ```

## Summary

**ProductTopology** is central to topology:

- **Basis:** Products of component opens
- **Projections:** Always continuous and open
- **Separation:** All axioms inherited from components
- **Connectedness:** Inherited from all components
- **Compactness:** Inherited (Tychonoff!)
- **Convergence:** Component-wise convergence
- **Dimension:** Generally sum of component dimensions

### Key Theorems

| Theorem | Statement |
|---------|-----------|
| **Projection** | Projections are continuous and open |
| **Tychonoff** | Product of compact spaces is compact |
| **Separation** | Product is T_i iff all components are T_i (for i=0,1,2) |
| **Connectedness** | Product connected iff all components connected |
| **Convergence** | Convergence is coordinate-wise |
| **Metrizability** | Finite product of metrizable is metrizable |
